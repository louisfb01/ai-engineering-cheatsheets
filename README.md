# AI Engineering Cheatsheets

Practical, decision-ready cheatsheets for AI engineering, by [Louis-Francois Bouchard](https://www.linkedin.com/in/whats-ai/) ([X](https://twitter.com/Whats_AI), [YouTube](https://youtube.com/whats_ai), [other AI resources](https://links.louisbouchard.ai/)).

These cheatsheets give you actionable decision frameworks, not theory. Each one is designed to help you pick the right technique, architecture, or workflow for your AI project and get it right on the first pass.

## Cheatsheets


| Cheatsheet                                                                    | What it covers                                                                                                                                                             |
| ----------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [AI Engineering Playbook](AI_Engineering_Playbook.md)       | End-to-end decision frameworks: which AI technique to use, model selection, prompting strategies, RAG pipelines, memory patterns, evaluation, and production optimization. |
| [Agent Architecture Guide](Agent_Architecture_Guide.md)     | When to use workflows vs. single agents vs. multi-agent systems. Includes architecture selection questions, system design decisions, and core engineering rules.           |
| [Anti-Slop AI Writing Guide](Anti_Slop_AI_Writing_Guide.md) | How to get human-sounding output from any LLM. Includes a 7-section prompt template, a banned-word list, style and structure rules, and a two-model write-then-review workflow. |


## How to use

1. Pick the cheatsheet relevant to your current problem.
2. Use the decision tables to find your situation and follow the recommended approach.
3. For the Anti-Slop template, copy the complete template section directly into your LLM of choice.
